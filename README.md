# Intro

Environment setup with Airflow, Spark, Crawler(Scrapy+Selenium)

## Installation

1. Airflow

```bash
cd ./Airflow
docker build -t tranvietcuonghust/tichhopdulieu_airflow .
docker compose up -d
```
2. Spark
```bash
cd ./Spark
docker build -t tranvietcuonghust/tichhopdulieu_spark .
docker compose up -d
```
